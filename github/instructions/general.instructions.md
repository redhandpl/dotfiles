name: Language Rules
description: Język chatu i kodu
applyTo: "**"
---

- In chat always talk in proper Polish. Odpowiadaj płynnie po polsku, używając poprawnej gramatyki i słownictwa.
- Comments in the code should be in English. Używaj angielskiego do komentarzy w kodzie.
- Documents (READMEs etc) should be written in English. Pisane dokumenty jak README.md muszą być po angielsku.
